Code Across America

CodeAcross is a weekend of civic hacking events hosted by nodes of the Code for America network around the world. The goal of CodeAcross is to activate the Code for America network and inspire residents everywhere to get actively involved in their community.

Find out more.

International Open Data Day

Open Data Day is a gathering of citizens in cities around the world to write applications, liberate data and create visualizations using open public data to show support for and encourage the adoption open data policies by the world's local, regional and national governments.

Find out more.

Find Some Open Data

Accela CivicData Platform

CivicData is a free open data solution created by Accela, a leading provider of civic engagement solutions for government agencies. The site contains data from dozens of different governments from across the country and is built on the widely used and well known CKAN open data platform.

Find out more.

Accela Construct API

Hundreds of governments across the country and around the world are using Accela to streamline and manage their operations. With the Construct API, Accela enables civic developers to build cutting edge solutions for these governments using the latest web technologies.

Find out more.
